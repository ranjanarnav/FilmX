# 🚀 FilmX Features

FilmX is a Django-based movie streaming platform designed to simulate a real-world streaming system similar to Netflix.

---

# 🎬 Movie Management Features

- Upload movies using Django Admin
- Upload movie thumbnails
- Upload video files
- Add movie titles
- Add movie descriptions
- Assign categories to movies
- Edit existing movies
- Delete movies
- Organized movie storage

---

# 🎭 Category System

FilmX includes a category-based movie organization system.

Admins can:

- Create categories
- Edit categories
- Delete categories
- Assign categories to movies
- Browse movies by category

Example Categories:

- Action  
- Comedy  
- Drama  
- Horror  
- Sci-Fi  

---

# 🔍 Search System

Users can search movies easily.

Search supports:

- Dynamic movie listing
